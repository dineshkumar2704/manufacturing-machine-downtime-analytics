# 🏭 Manufacturing Machine Downtime Analytics System

## Overview

The Manufacturing Machine Downtime Analytics System is an AI-powered analytics application developed using Python and Streamlit. It helps analyze machine downtime, identify failure patterns, predict future failures, detect anomalies, forecast downtime trends, and generate AI-based maintenance recommendations.

This project demonstrates practical data analytics, machine learning, dashboard development, and predictive maintenance techniques using real-world manufacturing concepts.

---

## Features

- Machine Downtime Analysis
- Issue Analytics
- Cost Analytics
- Shift Performance Analysis
- Machine Failure Prediction
- Downtime Forecasting
- Anomaly Detection
- AI Maintenance Recommendations
- PDF Report Generation
- Interactive Streamlit Dashboard

## Dashboard Modules

### Overview

Displays overall manufacturing performance and important KPIs.

### Machine Analytics

Provides machine-wise downtime analysis and performance comparison.

### Issue Analytics

Analyzes major causes of downtime and recurring machine issues.

### Cost Analytics

Estimates downtime cost and production loss.

### Shift Analytics

Compares downtime across different production shifts.

### Forecasting

Predicts future downtime using historical trends.

### Predictions

Predicts machine failures using machine learning models.

### Anomaly Detection

Detects abnormal machine behavior using AI.

### AI Recommendations

Generates maintenance recommendations and exports PDF reports.

## Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/manufacturing-machine-downtime-analytics.git
```

Move into the project folder

```bash
cd manufacturing-machine-downtime-analytics
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```

or

```bash
python -m streamlit run app.py
```
